## REST API (MVC Model) for Dashboard

- Available on [Link](https://rahulsai.live/api/)
- Documentation on [Postman](https://documenter.getpostman.com/view/5649815/SzYdRvab)

- To start the server

```sh
npm install

# development
npm start

# production
npm install pm2 -g
pm2 start npm -- start
```
